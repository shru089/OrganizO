# 🌿 OrganizO

> **Structure your life, gently.**

🌎 **Live App**: [https://organizolife.netlify.app/](https://organizolife.netlify.app/)

OrganizO is a premium, calm productivity application featuring a Sunlit Bamboo Forest aesthetic. It is designed for focus, mental clarity, and an immersive user experience.

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **🎋 Sunlit Forest Theme** | High-quality bamboo forest background with interactive "firefly" light motes. |
| **🪟 Glassmorphic Dashboard** | A premium `app.html` workspace with a sidebar, adaptive greetings, and functional widgets. |
| **⏱️ Zen Focus Timer** | Built-in Pomodoro logic with a nature-inspired palette. |
| **✨ Micro-Interactions** | Gentle breathing buttons, floating cards, and glowing text effects. |
| **📱 Responsive Design** | Mobile-optimized layout with a focus on readability and smooth transitions. |

---

## 🎨 Design Philosophy

OrganizO is built on the principle of **Calm Productivity**. Every design choice — from the emerald green tones to the slow easing functions — is intended to reduce stress and invite focus. 

OrganizO intentionally avoids a backend to maximize privacy, reduce complexity, eliminate server costs, and allow offline usage. 

---

## 🛠️ Tech Stack

- **Core**: Vanilla HTML5, CSS3, JavaScript
- **Styling**: Backdrop Filters, Flexbox, Grid
- **Typography**: Playfair Display, Inter, Outfit (Google Fonts)
- **Deployment**: Netlify Ready

## 🚀 Getting Started

1. **Explore**: Open `index.html` to see the public landing page.
2. **Focus**: Click **"Try OrganizO"** to enter the workspace (`app.html`).
3. **Customize**: Modify colors and animations in `styles.css` and `app-styles.css`.
